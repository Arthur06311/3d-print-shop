# Créditos dos Modelos 3D

## IMPORTANTE: Licença Comercial
Antes de vender qualquer modelo, verifique se o criador permite **uso comercial**.
Muitos modelos são apenas para uso pessoal. Verifique a licença na página de cada modelo.

---

## Como adicionar as fotos ao site

1. Acesse cada link abaixo
2. Baixe/salve a foto principal do modelo (clique com botão direito > "Salvar imagem como...")
3. Salve na pasta `images/` com o nome indicado
4. As fotos aparecerão automaticamente no site
